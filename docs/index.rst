RadVel: |br| The Radial Velocity Fitting Toolkit
==============================================

Welcome to the documentation for ``radvel``, a Python package for
modeling of radial velocity time series data.

Please cite the `Fulton, Petigura, Blunt & Sinukoff (2018)  <http://adsabs.harvard.edu/abs/2018PASP..130d4504F>`_ and the following DOI if you make use of RadVel in your research.
